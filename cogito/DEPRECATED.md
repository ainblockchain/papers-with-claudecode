# DEPRECATED

The general Cogito container has moved to `ain-blockchain/cogito/`.

- **General infrastructure** (recipe watcher, x402 server, ain-client, base-chain): `ain-blockchain/cogito/`
- **Domain-specific recipes** (system prompts for paper enrichment, course generation): `papers-with-claudecode/cogito-recipes/`
- **Recipe registration script**: `papers-with-claudecode/scripts/register-recipes.ts`

The code in this directory is kept for reference but is no longer deployed.
The `deploy-cogito.yml` GitHub Actions workflow has been removed.
